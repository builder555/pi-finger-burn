## pi-finger-burn

A CPU Button state machine: it changes the state when you touch the CPU with your finger
Run it on a Pi (or any other system that outputs temperature)

# DO NOT USE ACTUAL FINGER!

you've been warned.

```bash
python3 burn.py
```
